Infinity-Flow

Trackable physical-digital coin toolkit – collectible coins that combine a physical metal shell with a nanosized nuclear core, leaf shielding, and cryptographic provenance.

View on GitHub Get Started

Concept

Shell

Metal Composition

Chosen metal (gold, silver, platinum…). Signal properties derived from material physics.

Core

Nuclear Fingerprint

Nanosized radionuclide (Ra-223/224) in an encapsulated carrier. SHA-256 fingerprint → unique tamper-proof ID.

Shield

Leaf Containment

Gold-leaf or graphene-gold composite at ~100 nm. Containment by default; configurable pulse window.

Provenance

Cryptographic Chain

Serial number stamped on coin. Immutable chain of minted / transferred / authenticated events.

Installation

# Clone and install in editable mode
git clone https://github.com/www-infinity4/Infinity-Flow.git
cd Infinity-Flow
pip install -e .

CLI Usage

# List all coins in the default set
infinity-flow list

# Filter by collectible tier
infinity-flow list --tier exotic

# Inspect a single coin (JSON output)
infinity-flow show gold

# Mint a full set and dump JSON
infinity-flow mint --minted-by "My-Mint"

# Verify the digital integrity of every coin
infinity-flow verify

# Simulate a signal pulse
infinity-flow pulse gold
infinity-flow pulse carbon_hybrid
